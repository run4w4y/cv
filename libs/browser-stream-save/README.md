# @cv/browser-stream-save

Browser runtime helpers for saving byte data.

The package wraps Blob-based downloads behind an Effect service so UI code can
save decrypted private files or generated bytes without owning browser download
details.

## Use

```ts
import { BrowserStreamSave, BrowserStreamSaveLayer } from '@cv/browser-stream-save'
import { Effect } from 'effect'

const program = BrowserStreamSave.pipe(
  Effect.flatMap((save) =>
    save.saveBytes({
      bytes,
      fileName: 'document.pdf',
      mimeType: 'application/pdf',
    })
  )
)

await Effect.runPromise(program.pipe(Effect.provide(BrowserStreamSaveLayer)))
```
