# @cv/private-content-config

Shared Effect config readers for private-content build and operator
environment variables.

This package centralizes the environment variable names used by the app,
content-build package, dev profile page, and private link tool.

## Variables

- `CONTENT_ROOT`: authored content repository root.
- `CONTENT_ID_SALT`: deterministic salt for opaque public ids.
- `PRIVATE_CONTENT_AUDIENCE_KEY`: key for reversible private audience path ids.
- `PRIVATE_CONTENT_ROOT_KEY`: 32-byte root key used to derive profile content
  keys.

`PRIVATE_CONTENT_ROOT_KEY` is the only private build secret read here. Derived
profile keys are generated by build/link tooling and are not environment inputs.

## Use For

- `readPrivateContentBuildConfigFromEnv()` for `CONTENT_ROOT` and
  `CONTENT_ID_SALT`.
- `readPrivateContentBuildSecretsFromEnv()` when private runtime artifacts are
  optional.
- `readRequiredPrivateContentBuildSecretsFromEnv()` for private-only tools.
- `missingPrivateContentAccessEnv()` for user-facing diagnostics.

## Verification

```bash
bunx nx run private-content-config:typecheck

```
